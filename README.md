# sql2struct
mysql database to golang struct for gorm model

# install
```shell
go install github.com/miming1688/sql2struct@latest
```



# usage
```shell
sql2struct --dsn="root:123456@tcp(localhost:3306)/test?charset=utf8mb4"
```
